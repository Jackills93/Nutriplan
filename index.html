Telegram configurato —
Come configurare (3 minuti):
1
Apri Telegram → cerca @BotFather → scrivi /newbot → copia il Bot Token.
2
Cerca @userinfobot → scrivi /start → copia il tuo Chat ID (numero).
3
Apri il tuo bot su Telegram e scrivi /start per attivarlo. Poi incolla qui sotto e clicca Salva.
Privacy
API key Anthropic, Bot Token e Chat ID sono salvati solo nel tuo browser (localStorage). Nessun dato viene trasmesso a server intermedi. Le chiamate AI vanno direttamente ad Anthropic, quelle Telegram direttamente a Telegram.
